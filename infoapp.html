VPN & YouTube Bypass

Ядро приватности и интеллектуальный обход региональных ограничений

Система построена на нативном android.net.VpnService и разделена на два изолированных микросервиса:

  • FoxVpnService: Управляет зашифрованными туннелями (WireGuard/OpenVPN). Реализует Kill-Switch, авто-переподключение при смене сети и маршрутизацию всего трафика через защищённые узлы.
  • YouTubeBypassService: Использует addAllowedApplication() для выборочного проксирования. Перенаправляет пакеты только целевых приложений (YouTube/Music), не влияя на глобальную скорость соединения.

Proxy Control

Динамическая агрегация MTProto узлов с автоматической валидацией

Модуль ProxyRepository собирает конфигурации из внешних источников в реальном времени:

  • Источники: FoxDen Core & Telegram Proxy Collector. Парсинг `mtproto://` ссылок с валидацией формата.
  • Health-Check: Асинхронный пул корутин (bulkHealthCheck) замеряет задержку через TCP-сокеты. Узлы сортируются по пингу, оффлайн-серверы скрываются.
  • Кэширование: Результаты сохраняются в SharedPreferences с TTL 60 минут для мгновенного запуска UI.

DNS Ecosystem

Умная маршрутизация запросов и защита на уровне системы

Интеграция с PrivateDns API и кастомным маппером DnsMapper:

  • DoT Шифрование: DNS-over-TLS скрывает запросы от провайдера. Автоматическая ротация между Cloudflare, Google и AdGuard.
  • Profile Mapping: Маппинг IP → Профиль (Supercell, Xbox, Gaming). Глубокая настройка через системные интенты без root-доступа.
  • Фильтрация: Блокировка рекламных доменов и трекеров на уровне резолвера перед отправкой пакетов.

AI Hub

Единый шлюз к нейросетям с сохранением локального контекста

Архитектура WebView Shell объединяет Gemini, DeepSeek, Qwen, Claude 3.5 и Arena AI:

  • Бесшовный UI: Инъекция JS для принудительной тёмной темы и адаптации под Compose-окружение.
  • Room Database: ChatRepository и ChatDao хранят историю сессий, метаданные и роли сообщений локально. Полная конфиденциальность без облачной синхронизации.
  • State Flow: Реактивное обновление интерфейса через Kotlin Flow. Поддержка вложений и генерация прямых ссылок с предзаполненным промптом.

Технологический стек

Современная Android-архитектура для максимальной производительности

Приложение разработано с соблюдением лучших практик Android-разработки 2024-2026:

  • UI: 100% Jetpack Compose. Декларативный рендеринг, Material 3, кастомные темы с `CompositionLocal`.
  • Сеть: Retrofit + OkHttp для REST-запросов, парсинг JSON/Base64 подписок, интерцепторы логирования.
  • Асинхронность: Kotlin Coroutines & Flow. Управление жизненным циклом через `viewModelScope` и `LaunchedEffect`.
  • Целевая платформа: API 34 (Android 14+), компиляция под JVM 17, оптимизация под R8/ProGuard.
